# Vida que Prospera — Landing Page

Landing page da mentoria **Vida que Prospera** de Vanessa Rocha (@vanessarochabr).

## Stack
- HTML estático + Tailwind CDN
- Fontes: Noto Serif + Manrope (Google Fonts)
- Ícones: Material Symbols Outlined
- Deploy: Vercel

## Deploy
1. Import no Vercel apontando para este repo
2. Framework Preset: **Other**
3. Build Command: (deixar vazio)
4. Output Directory: `.`
5. Deploy

## Checkout
Substituir `SEU-LINK-AQUI` pelo link do checkout Hotmart em `index.html`.

Buscar e substituir: `https://pagar.me/checkout/SEU-LINK-AQUI`
Substituir por: `https://pay.hotmart.com/SEU_CODIGO`

## Domínio
Após deploy, conectar domínio customizado nas Settings do projeto no Vercel.
